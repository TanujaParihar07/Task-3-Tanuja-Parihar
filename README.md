# Customer Segmentation using PCA and K-Means Clustering

## Objective

The objective of this project is to identify different customer groups based on demographic and spending behavior using unsupervised machine learning techniques.

The project applies PCA for dimensionality reduction and K-Means Clustering for customer segmentation.

- K-Means Clustering
